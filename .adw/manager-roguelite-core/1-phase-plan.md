# Phase Plan: manager-roguelite-core

## Task

Design and prototype the core systems of a basketball manager roguelite game concept documented in `docs/brainstorming/manager-roguelite/1-brainstorming.mdx`. The work has two stages: first, resolve the critical open design questions across the three core systems (tactic chain / playbook construction, chemistry & familiarity economy, nemesis / rivalry mechanic) and produce a concise design spec (PRD) that closes those questions. Second, build a minimal runnable code prototype that exercises the preparation-then-execution autobattler loop — validating that the core mechanic is fun and legible before committing to full implementation.
